关于前端小学期

在上这个小学期之前，我也不能算完全没有接触过前端的内容。之前的离散2大作业就用自己搭建了一个网站展示用图论算法对豆瓣电影数据集的分析，至少给我的感觉是好像也没那么难，刚刚上手时学点基本的规则就能够依葫芦画瓢地做出一些东西来。

不过今天的课的确给了我一些小小的惊讶。最开始老师讲变量类型的时候，讲到三种不同的声明方式：

  • var
  • let
  • const

我一直也分不清这三个，自己之前全都是从头到尾 var 的，也对此不感兴趣；但是当他讲到var 的生命周期造成的问题时，我才意识到这好像挺严重的，因为出现问题的异步调用的函数我以前从来没有接触过，甚至完全不知道有这种东西的存在，但如果真的哪一天我无意中用到了，恐怕debug会费相当大的力气。就这一件事，我意识到并不能轻松他，不管是html、css还是javascript，还是应该把基本功打的扎实一些。

因此，我也对这门课产生了比“练手而已”更深的期待：

  • 语法的基础打牢，踏实地把基本的规则掌握，少犯课堂上例子那样愚蠢的错误；
  • 掌握常用框架的使用，如 jQuery、vue.js 等；
  • 体验前端开发完整的流程，熟练使用GitHub与人合作项目。

总之，既然要学这门课，还是希望自己能少一点浮躁，不要轻视这些技术，至少作为一名训练中的程序员还是要技能树还是应该广一点。
